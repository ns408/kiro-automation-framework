WORKFLOW EXAMPLES:

1. READ-ONLY EXPLORATION (Level 0)
   User вҶ’ TrustLevel(0) вҶ’ Whitelist вҶ’ Read Files вҶ’ Logger

2. SAFE FILE CREATION (Level 1)
   User вҶ’ TrustLevel(1) вҶ’ Whitelist вҶ’ Create Files вҶ’ Logger

3. MODIFY WITH BACKUP (Level 2)
   User вҶ’ TrustLevel(2) вҶ’ Whitelist вҶ’ Backup вҶ’ Modify вҶ’ Logger

4. SANDBOX TESTING (Level 2-4)
   User вҶ’ Sandbox.create() вҶ’ Copy Project вҶ’ Run Operations вҶ’ 
   Diff вҶ’ Review вҶ’ Apply/Discard вҶ’ Logger

5. ACTION SCRIPTS (Level 3)
   User вҶ’ TrustLevel(3) вҶ’ ActionRunner.generate() вҶ’ 
   Human Review вҶ’ run-action.sh вҶ’ Logger

6. FULL AUTOMATION (Level 4)
   User вҶ’ TrustLevel(4) вҶ’ Whitelist вҶ’ ActionRunner.execute() вҶ’ Logger
